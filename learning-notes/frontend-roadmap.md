# Frontend Foundation Roadmap

This note records my current frontend learning plan.

## Stage 1: HTML and CSS

Goal: build static pages with semantic HTML and responsive CSS.

Practice:
- Personal profile page
- Responsive layout exercises
- Rebuild simple website sections

## Stage 2: JavaScript and DOM

Goal: add common interactions to web pages.

Practice:
- Theme switching
- Form validation
- Todo list
- Simple quiz page

## Stage 3: ES6 and Modular Code

Goal: write cleaner and more maintainable JavaScript.

Practice:
- Split code into small functions
- Use `const` and `let` correctly
- Learn arrays, objects, modules, and async basics

## Review Habit

For each small project, I should record:
- What problem I solved
- Why I chose this approach
- What bug I met
- How I fixed it
